CSS INTERVIEW QUESTIONS

Day 1

  1. Difference between flex row and column?

    2. Flex are used for creating one dimensional layout.All child elements are arranged in main axis.
    By Default Flex-Direction is row that means all elements will be arranged horizontally.
    When Flex-Direction is column, all child elements will be arranged vertically.

  2. Explain inline,internal and external stylesheet?

    3. Inline stylesheet:

    Placed the CSS inside an HTML tag and affect only that element.

    Internal stylesheet:

    Placed inside a style tag which goes inside <head> of HTML document.

    External stylesheet:

    Exists in a separte file called an External stylesheet and requires a <link> element placed into the head section of HTML file.

  3. Justify-content allows you to do what?

    4. It defines how the browser distributes space between and around content items along the main-axis of flex container and inline axis of Grid container.

  4. Difference between Absolute and Relative Positioning?

    5. Relative:

    The element is positioned relative to its normal position.Also, it has space reserved for the content.

    Absolute:

    The element is absolutely to its first positioned parent. Also,it has no space reserved for the absolute element.

  5. What is grid-template-columns used for?

    6. It specifies number of columns in the Grid layout.

  6. What is z-index in CSS?

    7. It is CSS property that defines the order of overlapping HTML elements.Elements with higher index will be placed on top of elements with lower index.

  7. What is the Difference between Padding and Margin?

    8. Padding creates extra space within an element.

    Margin creates extra space around an element.

Day 2

  1. What is box-sizing?

    2. It allows us to include Padding and border in an element's total width and height. The box-sizing property sets how the total width and height of an element is calculated.

  2. What is Animation delay?

    3. It specifies amount of time to wait from applying animation to an element before beginning to perform animation.

  3. Which property you will use to merge cells vertically in grid?

    4. CSS grid-column property

  4. Which property you will use to merge cells horizontally in grid?

    5. Grid span property

  5. What is the Difference between display none and visibility hidden?

    6. Visibility hidden hides the element but it still takes up space in the layout.

    Display none removes the element from the document .It does not take up any space.

  4. What are CSS Combinators?

    5. A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.

    There are four different combinators in CSS:
    • descendant selector (space)
    • child selector (>)
    • adjacent sibling selector (+)
    • general sibling selector (~)

  5. What are Pseudo selectors in CSS?

    6. A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s). For example, the pseudo-class :hover can be used to select a button when a user's pointer hovers over the button and this selected button can then be styled.
